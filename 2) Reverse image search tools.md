# Reverse image search tools — OSINT (Person identification)
Tools focused on identifying people, tracking digital footprints, and linking images to online identities.  
These tools are commonly used in OSINT investigations to verify identities, detect fake profiles, or map a person’s online presence.

## Tools

- **Google Images**  
  ― find websites, social networks, and articles where a person’s photo appears  
  Useful to identify reused profile pictures across platforms  
  https://images.google.com

- **Yandex Images**  
  ― one of the most powerful reverse image search engines for faces  
  Often returns social media profiles and less-indexed websites  
  https://yandex.com/images

- **Bing Visual Search**  
  ― helps locate similar face images and linked web pages  
  Useful for cross-checking results from Google/Yandex  
  https://www.bing.com/images

- **TinEye**  
  ― track exact matches of a person’s image and see where it was first published  
  Useful to detect stolen or reused profile pictures  
  https://tineye.com

- **PimEyes**  
  ― face recognition search engine designed to find appearances of a person across the web  
  Very effective for locating social media profiles and public photos  
  https://pimeyes.com

- **FaceCheck.ID**  
  ― facial recognition OSINT tool specialized in identifying people across multiple sources  
  Useful for deep identity investigations  
  https://facecheck.id

- **Social Catfish**  
  ― identity verification tool combining reverse image search and social media lookups  
  Often used to detect fake profiles or scams  
  https://socialcatfish.com

## Techniques

- Crop the image to focus only on the face before uploading  
- Run the same image across multiple tools (results vary greatly)  
- Check usernames, bios, and background details from matched profiles  
- Cross-reference with social networks, forums, and public databases
